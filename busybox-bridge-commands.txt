TRY THESE COMMANDS IN TERMUX (using built-in tools):

1. Check if busybox netcat is available:
busybox nc --help

2. If busybox works, create the bridge:
echo -e "HTTP/1.1 200 OK\r\nContent-Type: application/json\r\nAccess-Control-Allow-Origin: *\r\n\r\n{\"status\":\"connected\",\"model\":\"qwen2.5-1.5b-instruct\"}" | busybox nc -l -p 8080

3. Check available system tools:
ls /system/bin/ | grep -E "(nc|netcat|curl|wget|socat)"

4. Alternative - check Termux built-ins:
which nc netcat curl wget socat
